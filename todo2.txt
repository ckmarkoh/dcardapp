未登入狀態

	用FB登入  ->       輸入註冊ID   ->   登入狀態(背景是相機畫面)
		    (如果第一次用FB帳號登入)

	用信箱登入   ->   Email登入  ->   登入狀態(背景是相機畫面)

		     ->   Email註冊  ->   輸入信箱密碼註冊   ->     輸入註冊ID   ->   登入狀態(背景是相機畫面)      
   		       (如果尚未註冊)      (或可用FB登入)


登入狀態  ->   拍照完   ->  傳送訊息  ->   朋友列表  ->   送出  ->   登入狀態(背景是相機畫面)
	     (點相機照相) 


訊息列表
      
      箭頭(傳送人)

	  顯示 1.傳送給誰 
	       2.對方是否看過
	  時間 1.送出的時間

      星星(新訊息)

	  顯示 1.傳送者的名字粗體
	  時間 1.收到的時間
	  打開方式  :   按住人名不放才能看訊息，放掉看不到，但繼續倒數，時間一到就再也打不開


      流星(打開新訊息後變流星)
	  顯示 1.傳送者的名字變沒有粗體
	  時間 1.收到的時間

      清除(全部訊息清除，不管有沒有新訊息)


功能列表

      朋友邀請   ->  好友邀請列表   ->   返回功能列表
					(處理完好友邀請，按箭頭才返回功能列表)

      加朋友   ->  加朋友畫面   ->   返回功能列表
				      (輸入好友ID，加完好友就返回  或  按箭頭返回)
